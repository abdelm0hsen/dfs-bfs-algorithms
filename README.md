# DFS / BFS Algorithms

A graph traversal project that demonstrates Breadth-First Search (BFS) and Depth-First Search (DFS) on a sample graph.

## Features

- Graph implementation with adjacency list
- BFS traversal using queue
- DFS traversal using stack
- Interactive start-node selection

## Project Structure

```
dfs-bfs-algorithms/
  main.py
  requirements.txt
  README.md
```

## Run

```bash
python main.py
```

## Learning Outcomes

- Graph data structures
- Traversal strategy differences
- Algorithmic problem-solving basics
